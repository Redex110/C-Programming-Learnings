# Sum of Harmonic Series

## Purpose
This program calculate the sum of harmonic series that is given below for the value N.

	Hamonic Seies : 1 + 1/2 + 1/3 +........+ 1/N.

## Input
- The value for the N from the range 1 to 99.

## Output
- The sum of harmonic series for the value of N.

##Example


## How to Compile
```bash
gcc main.c -o program
```

## How to Run
```bash
./program
```
