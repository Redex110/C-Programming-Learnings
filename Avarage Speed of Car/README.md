# Average Speed of Car

## Purpose
This program calculates the average speed of a car over 5 time intervals.
It displays a table showing the time (in hours) and distance (in KM) for each interval,
then prints the overall average speed in KMH.

## Input
- No user input required.
- Distance values are predefined: `{10, 20, 45, 69, 75}` KM over 5 hours.

## Output
- A formatted table displaying time (H) and distance (KM) for each interval.
- The calculated average speed of the car in KMH.

Example output:
```
--------------------------------
      Average Speed of Car
--------------------------------
   |   Time   |   Distance  |
   |   1.0H   |   10.00 KM  |
   |   2.0H   |   20.00 KM  |
   |   3.0H   |   45.00 KM  |
   |   4.0H   |   69.00 KM  |
--------------------------------
    Average Speed : 17.25 KMH
--------------------------------
```

## How to Compile
```bash
gcc main.c -o program
```

## How to Run
```bash
./program
```
