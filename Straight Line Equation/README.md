# Straight Line Equation

## Purpose
This program takes three integer coefficients (`a`, `b`, `c`) from the user representing a straight line in the form **ax + by = c**, then displays the line equation along with its **slope**, **x-intercept**, and **y-intercept**.

## Input
Three integers `a`, `b`, and `c` — the coefficients of the line equation `ax + by = c`.

## Output
- The equation of the line
- Slope of the line: `-(a/b)`
- X-intercept: `c/a`
- Y-intercept: `c/b`

**Example:**
```
----------------------------
Straight Line Equation
----------------------------
Enter the value for a : 2
Enter the value for b : 4
Enter the value for c : 8
----------------------------
Equation of Line : 2x + 4y = 8
Slop of Line : -0.50
X-Intersept : 4.00
Y-Intersept : 2.00
----------------------------
```

## How to Compile
```bash
gcc main.c -o program
```

## How to Run
```bash
./program
```
